# Digital Democracy Toolkit

Selamat datang di repositori **Digital Democracy Toolkit**! Toolkit ini dikembangkan oleh konsorsium LSM KOMPAS, FISIP Unjani, dan FDKV Universitas Widyatama, didukung oleh Tifa Foundation melalui program Digital Democracy Initiative.

## 📦 Isi Repositori

- **01-policy-toolkit/** - Alat analisis kebijakan (template dan panduan)
- **02-communication-toolkit/** - Template komunikasi visual dan panduan keamanan
- **03-video-tutorials/** - Tautan video tutorial penggunaan toolkit
- **04-project-playbook/** - Panduan replikasi model kolaborasi (segera hadir)

## 🚀 Cara Menggunakan

1. Kunjungi folder yang sesuai dengan kebutuhan Anda.
2. Unduh file dalam format yang diinginkan (PDF, DOCX, PPTX).
3. Baca panduan yang tersedia di setiap folder.
4. Untuk akses materi terbatas (Companion Pack), hubungi kami melalui [website](https://toolkit.lsmkompas.id).
